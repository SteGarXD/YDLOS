# Диагностика: почему не применяются стили сводной таблицы

## Возможные причины

### 1. Сборка не перезапущена
**Решение:** После любых изменений в `components/datalens-ui` нужно выполнить полную пересборку:

```powershell
cd D:\YDLOS\datalens
.\build-and-run-ui.ps1
```

Скрипт удаляет `dist`, собирает заново, строит Docker-образ и перезапускает контейнеры. Без этого используется старый образ.

### 2. pivotFallback = 'on'
При включённом pivot fallback используется **old-pivot-table**, а не backend-pivot-table. Наши стили заданы для backend-pivot-table.

**Проверка:** В редакторе чарта → Настройки → «Режим отображения сводной» (или аналогичная опция) — должно быть **выключено** / «Новый режим».

### 3. Итоги отключены
Футер (блок «ИТОГО» внизу) появляется только при включённых итогах.

**Проверка:** В настройках сводной таблицы включите «Итоги» / «Grand totals».

### 4. Другой компонент таблицы
Если `chartkitType !== 'table'`, используется старый виджет таблицы (не наш Table).

**Проверка:** В DevTools (F12) найдите таблицу. Должны быть классы:
- `dl-table`
- `dl-table__footer`
- `dl-table_prepared`

Если видите `chartkit-table` без `dl-table` — используется старый виджет.

### 5. Кэш браузера
**Решение:** Жёсткое обновление: Ctrl+Shift+R (или Cmd+Shift+R на Mac). Либо откройте в режиме инкогнито.

### 6. Футер не виден из‑за пагинации
Футер показывается только когда `isEndOfPage === true` (последняя страница при пагинации).

**Проверка:** Прокрутите таблицу до конца или перейдите на последнюю страницу.

---

## Быстрая проверка в DevTools

1. Откройте сводную таблицу на дашборде.
2. F12 → вкладка Elements.
3. Найдите `tfoot.dl-table__footer` или `td` внутри футера.
4. Проверьте:
   - Есть ли inline-стиль `border: 1px solid #000` или `border-color: #000`?
   - Есть ли класс `dl-table__td`?
   - Какой `border-color` в Computed?

Если inline-стили с чёрными границами есть, но визуально границы серые — возможно, другой CSS перекрывает (проверьте порядок загрузки стилей).

---

## Проверка API-ответа

Откройте Network → найдите запрос к `/api/run` или аналогичный. В ответе `data`:

- `data.footer` — должен быть массив строк с ячейками.
- В `data.footer[0].cells` у каждой ячейки может быть `css: { borderColor: '#000000' }` (бэкенд).

Если `data.footer` пустой — итоги отключены или используется old-pivot-table.
